## Before getting started

You have reached this file and for this, you deserve our warm thanks already!
This project is a piece of a larger project within Wiki Med (MDWiki) to bring in data from OurWorldInData (OWID) and create beautiful visuals to enhance our understanding of medecine related data. You will find more info here: https://mdwiki.org/wiki/WikiProjectMed:OWID
Thanks for your contributions!


## cloning

Create a fork and clone the repo. Make sure to create a branch for each feature or bug fix for clarity.

## testing

### quick version

If you want to test locally, open up demo.html in a web browser. This is meant to be a quick hack and does not replace a full test.

### full version

* Go to https://mdwiki.org/wiki/Main_Page and create an account if you haven't already, and log in
* Copy MediaWiki:Common.js from this repo to https://mdwiki.org/wiki/Special:MyPage/common.js 
* Copy MediaWiki:Common.css to https://mdwiki.org/wiki/Special:MyPage/common.css
* go to Special:Preferences, under gadgets, be sure that "Shows a popup dialog with a slider for OWID images" is not checked.
